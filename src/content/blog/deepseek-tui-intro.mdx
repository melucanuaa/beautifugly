### 自动模式（Auto Mode）

杀手锏功能。输入 `/model auto`，自动决定模型和推理强度：

<div style="background:var(--color-bg-card);border-radius:10px;padding:14px;margin:1rem 0;font-size:0.85rem">
  <div style="display:flex;align-items:center;gap:8px;margin-bottom:8px">
    <span style="background:var(--color-accent-secondary);color:#fff;padding:2px 8px;border-radius:99px;font-size:0.7rem">简单</span>
    <span>→ <code>flash</code>，不开推理，便宜快速</span>
  </div>
  <div style="display:flex;align-items:center;gap:8px">
    <span style="background:var(--color-accent);color:#fff;padding:2px 8px;border-radius:99px;font-size:0.7rem">复杂</span>
    <span>→ <code>pro</code>，高推理，质量优先</span>
  </div>
</div>

背后是轻量路由：flash 先判断任务难度，再分配资源。你只管干活。

### 推理强度

`Shift+Tab` 循环切换：**off**（不推理，简单问答）→ **high**（中等，一般编码）→ **max**（最强，架构设计/复杂调试）。

### 工作区回滚

每轮操作前后自动快照，`/restore` 或 `revert_turn` 即可回滚，不动 `.git`。相当于代码"后悔药"。

## 安装

```bash
npm install -g deepseek-tui   # 一行搞定
deepseek --version             # 验证安装
deepseek --model auto          # 启动自动模式
```

首次运行提示输入 DeepSeek API Key，保存到 `~/.deepseek/config.toml`。大陆用户 npm 慢可用 Cargo 镜像：

```bash
# ~/.cargo/config.toml
# [source.crates-io]
# replace-with = "tuna"
# [source.tuna]
# registry = "sparse+https://mirrors.tuna.tsinghua.edu.cn/crates.io-index/"

cargo install deepseek-tui-cli --locked
cargo install deepseek-tui --locked
```
